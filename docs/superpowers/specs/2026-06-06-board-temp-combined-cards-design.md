# 板块温度合并卡片

## 目标

将市场热度页面第一排的 4 个独立板块温度卡片（上证主板、科创板、深证主板、创业板）合并为 2 个组合卡片：
- **主板温度**：上证主板 + 深证主板
- **双创温度**：科创板 + 创业板

## 参考模式

领涨/领跌板块已采用的组合卡片模式：父卡片 + 内部两个子项并排。

## 设计

### 布局

第一排从 `repeat(6, 1fr)` 改为 `repeat(4, 1fr)`：

```
[🔥 市场温度] [⚠️ 压力指数] [📊 主板温度] [🚀 双创温度]
```

### 卡片结构

- **父卡片**：固定中性灰渐变背景，标题 + 两个子项 flex 并排
- **子项**：各自显示板块名、温度分数、等级，背景随自身温度等级变色（半透明叠加），hover 加深
- **交互**：子项独立可点击，打开对应板块的历史趋势弹窗；hover 显示 3 维度 tooltip

### 影响范围

仅修改 `frontend/src/components/market-heat/TemperatureCard.tsx`，数据层和 API 不变。
