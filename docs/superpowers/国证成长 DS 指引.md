## 二、官方全量查询（免费且最准确）

国证成长100指数由**国证指数公司**编制，在官网可以免费查到最权威的完整成分股列表：

1. 访问 **国证指数官网**：`http://www.cnindex.com.cn`
2. 在搜索框输入指数名称"**国证成长100**"或指数代码"**980080**"
3. 进入详情页后，选择"**样本股数据**" → "**样本股清单**"，即可查看全部100只成分股、行业分类及权重信息

官网同时提供以下官方正式文件可下载：
- **编制方案**：`http://www.cnindex.com.cn/docs/gz_980080.pdf`
- **单张页（含基本信息）**：`http://www.cnindex.com.cn/html2pdf/preview/jj_980080.pdf`

这是获取完整、权威成分股名单的首选渠道。

## 三、编程自动获取（Python方案）

如果需要定期（如每季度指数调样后）自动更新成分股名单，可通过以下Python接口批量拉取。

### 方案一：AkShare（完全免费，推荐）

AkShare的国证指数库提供了样本详情、历史样本和历史调样等接口：

```python
import akshare as ak

# 获取国证指数所有列表
index_df = ak.index_cni_all()

# 获取国证成长100（980080）的历史样本详情
# 可传入 index='980080' 获取完整权重列表
constituents = ak.index_cni_hist(index='980080')
print(constituents)
```

支持的功能：
- 获取成分股代码、名称、权重
- 获取历史调样记录
- 分季度查询最新成分股名单

### 方案二：tushare 或其他开源库

tushare的指数成分股接口同样可以获取，需要相应积分权限：

```python
import tushare as ts

# 设置token
ts.set_token('your_token')

# 获取指数成分股
df = ts.index_member(index_code='980080')
print(df)
```

> tushare免费版有调用次数限制，获取全量数据可能需要一定积分。
